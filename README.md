# MERN Ecommerce Project

A full stack ecommerce application built using the MERN stack.

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Redux Toolkit
- Tailwind CSS

## Features

- User Authentication
- Product Listing
- Add to Cart
- Order Management
- Admin Dashboard

## Installation

Clone repository

git clone https://github.com/somesh-borade/ECOM.git

Backend Setup

cd backend
npm install
npm run dev

Frontend Setup

cd frontend
npm install
npm run dev
